射线光学模拟

自由、开源的网页程序，用于建立及模拟二维几何光学场景。

本项目，包括作品集内容，皆采用 Apache License 2.0 授权条款，且官方版本位于 PhyDemo（phydemo.app）网站（原先于 ricktu288.github.io）。本项目源代码公开于此 GitHub Repo，欢迎您贡献本项目。若有问题或建议，您可以在 GitHub 上建立 Issue 或 Discussion（请使用英文），也可以传送电子邮件至 ray-optics@phydemo.app（可使用中文）。
